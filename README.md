# Computer Graphics Lab Projects

This repository contains OpenGL-based computer graphics programming exercises and lab implementations using C++ and GLUT/FreeGLUT.

## Project Structure

```
├── lab1/              # Lab 1 implementations
│   ├── resolution.cpp # Screen resolution utilities
│   └── sushma.cpp     # Basic graphics operations
├── lab2/              # Lab 2 implementations
│   ├── BLA.cpp        # Bresenham's Line Algorithm
│   ├── DDA.cpp        # Digital Differential Analyzer
│   ├── lineGraph.cpp  # Interactive line graph with colored points
│   ├── midpointcircle.cpp # Midpoint Circle Drawing Algorithm
│   └── piechart.cpp   # Pie chart visualization
```

## Features

### Lab 1

- **Resolution Display**: Screen resolution detection and display
- **Basic Graphics**: Fundamental graphics operations

### Lab 2

- **Line Drawing Algorithms**: Implementation of DDA and Bresenham's algorithms
- **Circle Drawing**: Midpoint circle algorithm implementation
- **Interactive Visualizations**:
  - Line graphs with colored data points
  - Pie charts for data representation
